# GS1-128

GS1-128 extends Code 128 with Application Identifiers (AIs) for supply chain data.

## Usage

```ts
import { barcode } from "etiket";

// Parenthesized AI format (recommended)
barcode("(01)12345678901234(17)260101(10)ABC123", { type: "gs1-128" });

// GTIN only
barcode("(01)12345678901234", { type: "gs1-128", showText: true });
```

## Application Identifiers

| AI     | Name             | Length         | Type     |
| :----- | :--------------- | :------------- | :------- |
| `01`   | GTIN             | 14 digits      | Fixed    |
| `02`   | GTIN of contents | 14 digits      | Fixed    |
| `10`   | Batch/Lot        | up to 20 chars | Variable |
| `11`   | Production date  | YYMMDD         | Fixed    |
| `15`   | Best before      | YYMMDD         | Fixed    |
| `17`   | Expiry date      | YYMMDD         | Fixed    |
| `20`   | Variant          | 2 digits       | Fixed    |
| `21`   | Serial number    | up to 20 chars | Variable |
| `30`   | Variable count   | up to 8 digits | Variable |
| `310x` | Net weight (kg)  | 6 digits       | Fixed    |
| `37`   | Number of units  | up to 8 digits | Variable |

FNC1 separators are automatically inserted between variable-length fields.

## Raw Encoder

```ts
import { encodeGS1128 } from "etiket";

const bars = encodeGS1128("(01)12345678901234(10)ABC");
```
